这个命令可以帮助你看到最近到最远的提交日志.

同时，Git还提供了一个命令可以查看你之前用过的每个命令

在 git log 中，如果你 commit 文件了，你可以看的在 commit 后面跟了一串哈希数值，他是版本号(commit id),
当你觉得这次的修改错了，你要回退到上一次修改，你可以根据版本号去回退.

当然你可以不用写这么一大串，只写前面几个 Git 会帮你找到的.

使用 git reflog 命令，你会发现其中有个 HEAD 参数，这个可以表示版本 – 当前版本 HEAD@{0} 或 HEAD,
上一个版本 HEAD@{1} 或 HEAD^ , 一直到后面.你也可以用 HEAD 这个参数来版本回退的.

这就是回退到上一个版本.
